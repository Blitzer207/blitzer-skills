## TGInit

Module path: `flow_kit.library.test_guide.TGInit`

> Block to initialize connection to test.guide.

**Parameter:**

- TGInit.**PAR\_\_ENV_AUTH_KEY**

  - type: `str`

  - optional, default = `TEST_GUIDE_AUTH_KEY`

  - name of environment variable which holds auth_key, leave unset to use authentication key of flow trigger user.

- TGInit.**PAR\_\_PROJECT_ID**

  - type: `int`

  - optional, default = `0`

  - test.guide project_id, leave unset to use settings from resource adapter.

- TGInit.**PAR\_\_URL**

  - type: `str`

  - optional, default = \`\`

  - url to test.guide, leave unset to use settings from resource adapter.

**Returns:**

> config for test.guide connection

**Return type:**

> `flow_kit.tools.test_guide.test_guide_config.TestGuideConfig`
