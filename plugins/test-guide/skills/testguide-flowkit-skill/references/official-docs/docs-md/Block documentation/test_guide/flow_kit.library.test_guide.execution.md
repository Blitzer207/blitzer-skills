## ExecutePlaybook

Module path: `flow_kit.library.test_guide.execution.ExecutePlaybook`

> test.guide ExecutionApi: Execute a playbook. This will create execution tasks or a project job.

**Parameter:**

- ExecutePlaybook.**PAR\_\_PLAYBOOK_EXECUTION_REQUEST**

  - type: `tg_api_clients.execution.PlaybookExecutionRequest`
  - Description for the playbook.

- ExecutePlaybook.**PAR\_\_PLAYBOOK_ID**

  - type: `int`
  - ID of the playbook in test.guide

- ExecutePlaybook.**PAR\_\_CHECK_LAST_EXECUTION**

  - type: `bool`

  - optional, default = `None`

  - Only start a new execution if the last one has already been finished.

- ExecutePlaybook.**PAR\_\_CRON_EXPRESSION**

  - type: `str`

  - optional, default = `None`

  - Cron expression for a recurring execution in the UNIX cron format.

- ExecutePlaybook.**PAR\_\_CRON_TIME_ZONE**

  - type: `str`

  - optional, default = `None`

  - Time zone ID of the cron expression. Default value is 'UTC'.

- ExecutePlaybook.**PAR\_\_START_DATE**

  - type: `datetime.datetime`

  - optional, default = `None`

  - Start date and time of the execution for a delayed or recurring execution

- ExecutePlaybook.**PAR\_\_TG_CONFIG**

> test.guide ExecutionApi: Create an execution task.
>
> If a start date or a cronExpression is given a job is created.

**Parameter:**

- PostBundle.**PAR\_\_EXECUTION_TASK**

  - type: `tg_api_clients.execution.ExecutionTask`
  - Bundle description for the execution task.

- PostBundle.**PAR\_\_CHECK_LAST_EXECUTION**

  - type: `bool`

  - optional, default = `None`

  - Only start a new execution if the last one has already been finished.

- PostBundle.**PAR\_\_CRON_EXPRESSION**

  - type: `str`

  - optional, default = `None`

  - Cron expression for a recurring execution in the UNIX cron format.

- PostBundle.**PAR\_\_CRON_TIME_ZONE**

  - type: `str`

  - optional, default = `None`

  - Time zone ID of the cron expression. Default value is 'UTC'.

- PostBundle.**PAR\_\_START_DATE**

  - type: `datetime.datetime`

  - optional, default = `None`

  - Start date and time of the execution for a delayed or recurring execution

- PostBundle.**PAR\_\_TG_CONFIG**

  - type: `flow_kit.tools.test_guide.test_guide_config.TestGuideConfig`

  - optional, default = `None`

  - Config for connection to test.guide. If left unspecified, a default config will be used, containing the test.guide url and project ID of the ResourceAdapter as well as an authentication key of the user specified on the flow trigger. A custom test.guide config can be created by block TGInit.

**Returns:**

> Information about created task or job.

> `tg_api_clients.execution.CreateTaskResponse`
