## EtmFindProject

Module path: `flow_kit.library.tools.etm.project.EtmFindProject`

> Search for Project resources matching the provided fields query.

**Parameter:**

- EtmFindProject.**PAR\_\_ETM_CONFIG**

  - type: `flow_kit.tools.etm.etm.Etm`
  - ETM configuration created by block EtmInit.

- EtmFindProject.**PAR\_\_FIELDS_QUERY**

  - type: `str`

  - optional, default = `None`

  - The atom query string to filter resources or select fields (default: all resources and all fields).

- EtmFindProject.**PAR\_\_MAX_RESULTS**

  - type: `int`

  - optional, default = `None`

  - Maximum number of results to return (default: all results).

**Returns:**

> A list of matching Project objects.

**Return type:**

> `list[tracetronic_etm_client.model.qm.Project]`

## EtmReadProject

Module path: `flow_kit.library.tools.etm.project.EtmReadProject`

> Read a Project resource by its identifier URL.

**Parameter:**

- EtmReadProject.**PAR\_\_ETM_CONFIG**

  - type: `flow_kit.tools.etm.etm.Etm`
  - ETM configuration created by block EtmInit.

- EtmReadProject.**PAR\_\_RESOURCE_IDENTIFIER_URL**

  - type: `str`
  - The URL identifying the resource. It could contain oslc_config.context parameter. If so, the parameter value will be used as OSLC configuration for defining the context to work in and oslc_configuration argument should not be defined.

**Returns:**

> The deserialized Project object.

**Return type:**

> `tracetronic_etm_client.model.qm.Project`
