## JamaGetAbstractItems

Module path: `flow_kit.library.tools.jama.abstractitems.JamaGetAbstractItems`

> Search for items, test plans, test cycles, test runs, or attachments.

**Parameter:**

- JamaGetAbstractItems.**PAR\_\_JAMA_CONFIG**

  - type: `flow_kit.tools.jama.jama.Jama`
  - Created by block JamaInit.

- JamaGetAbstractItems.**PAR\_\_CONTAINS**

  - type: `list[str]`

  - optional, default = `None`

  - Filter on the text contents of the item. Strings taken literally. Multiple contains values will be bitwise ORed.

- JamaGetAbstractItems.**PAR\_\_CREATED_DATE**

  - type: `list[str]`

  - optional, default = `None`

  - Filter datetime fields after a single date or within a range of values. Provide one or two values in ISO8601 format (milliseconds or seconds) - yyyy-MM-ddTHH:mm:ss.SSSZ or yyyy-MM-ddTHH:mm:ssZ
