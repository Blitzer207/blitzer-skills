> ⚠ CAUTION: There is no stable pagination in the Polarion REST API by default. If the result set changes while querring all pages, the pages may overlap or skip resources. To ensure stable pagination, set the `revision` parameter to pin the result set to a specific version and set the `sort` parameter to a unique field. For more information, see [Polarion documentation](https://developer.siemens.com/polarion/basic-concepts.html#pagination-stability).
