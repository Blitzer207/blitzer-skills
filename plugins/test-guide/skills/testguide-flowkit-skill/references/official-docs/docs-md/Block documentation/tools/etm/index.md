## EtmInit

Module path: `flow_kit.library.tools.etm.EtmInit`

> Block for testing the connection to etm and for creating an etm configuration object for dependent blocks.

**Parameter:**

- EtmInit.**PAR\_\_URL**

  - type: `str`
  - url of IBM Engineering Test Management e.g. 'https://\<host\>:\<port\>/qm'

- EtmInit.**PAR\_\_ALLOW_INSECURE_CONNECTION**

  - type: `bool`

  - optional, default = `False`

  - NOT FOR PRODUCTION ENVIRONMENTS. Allows connections to unknown hosts, without performing ssl verification.

- EtmInit.**PAR\_\_ENV_KEY_BEARER_TOKEN**

  - type: `str`

  - optional, default = `None`

  - name of environment variable which holds a bearer access token

- EtmInit.**PAR\_\_ENV_KEY_PASSWORD**

  - type: `str`

  - optional, default = `None`

  - name of environment variable which holds password

- EtmInit.**PAR\_\_ENV_KEY_USER**

  - type: `str`

  - optional, default = `None`

  - name of environment variable which holds username

- EtmInit.**PAR\_\_RETRIES_CONFIGURATION**

  - type: `flow_kit.tools.retries_configuration.RetriesConfiguration`

  - optional, default = `None`

  - retries configuration overrides the default retry behavior

**Return type:**

> `flow_kit.tools.etm.etm.Etm`
