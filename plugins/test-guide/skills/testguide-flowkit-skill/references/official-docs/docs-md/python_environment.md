# Python environment

The flow bundle includes a Python environment which is used to execute the flow bundle.

Info

The Python interpreter is platform-specific. We provide Flow bundles with interpreters for Windows and Linux.

Currently, the environment uses Python version 3.12.

## Locations

The Python environment is located in the following folders:

- You can find the interpreter of the environment in the subdirectory `/interpreter/python.exe`.
- You can find the installed packages of the Python environment in the subdirectory `/site-packages`.
