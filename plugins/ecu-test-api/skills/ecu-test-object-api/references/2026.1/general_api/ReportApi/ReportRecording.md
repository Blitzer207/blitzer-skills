GetAbsolutePath()[¶](#ApiClient.ReportRecording.GetAbsolutePath "Link to this definition")  
Convenience method to get the absolute path to the recording file. Depending on the path type a base directory will be determined to make an absolute path:

- [`ReportRecording.PATH_REL_REPORT`](#ApiClient.ReportRecording.PATH_REL_REPORT "ApiClient.ReportRecording.PATH_REL_REPORT (Python attribute) — The constant used to specify that the recording path is stored relative to the report database."): The path to the current report will be used to make the path absolute.

- [`ReportRecording.PATH_ABS`](#ApiClient.ReportRecording.PATH_ABS "ApiClient.ReportRecording.PATH_ABS (Python attribute) — The constant used to specify that the recording path is stored as absolute path."): The path will be returned as it is.

- [`ReportRecording.PATH_REL_WORKSPACE`](#ApiClient.ReportRecording.PATH_REL_WORKSPACE "ApiClient.ReportRecording.PATH_REL_WORKSPACE (Python attribute) — The constant used to specify that the recording path is stored relative to the workspace used for the test."): The current workspace path will be used to make the path absolute.

Returns:  The absolute path

Return type:  str

GetMeasurementListFile()[¶](#ApiClient.ReportRecording.GetMeasurementListFile "Link to this definition")  
Returns the measurement list file associated with this recording, if any.

Returns:  The measurement list file or an empty string if no file is associated.

Return type:  str

GetNameAndFormatDetails()[¶](#ApiClient.ReportRecording.GetNameAndFormatDetails "Link to this definition")  
Returns the recording name and the format details (if set).

Returns:  The recording name and the format details as text of the form “\<name\> (\<formatdetails\>)” if format details are set else of the form “\<name\>”.

Return type:  str

Returns:  The number of signals or 0 if no measurement list file is associated.

Return type:  int

GetPath()[¶](#ApiClient.ReportRecording.GetPath "Link to this definition")  
Returns the path to the recording file. The path can be absolute or relative. The path type can be read by calling [`GetPathType()`](#ApiClient.ReportRecording.GetPathType "ApiClient.ReportRecording.GetPathType (Python method) — Returns how the path is to the recording is stored in the report.").
