# EdiabasArgument[¶](#ediabasargument "Link to this heading")

*class* EdiabasArgument[¶](#ApiClient.EdiabasArgument "Link to this definition")  
API facade of an Ediabas argument.

Clone()[¶](#ApiClient.EdiabasArgument.Clone "Link to this definition")  
Call this method if you want to use API-objects in multiple contexts, e.g. if you want to insert a specific test step in multiple blocks.

Returns:  A copy of this object

Return type:  [`EdiabasArgument`](#ApiClient.EdiabasArgument "ApiClient.EdiabasArgument (Python class) — API facade of an Ediabas argument.")

GetComment()[¶](#ApiClient.EdiabasArgument.GetComment "Link to this definition")  
Returns the comment of the argument.

Returns:  Comment of the argument

Return type:  str

GetName()[¶](#ApiClient.EdiabasArgument.GetName "Link to this definition")  
Returns the name of the argument.

Returns:  Name of the argument

Return type:  str

GetParameterExpression()[¶](#ApiClient.EdiabasArgument.GetParameterExpression "Link to this definition")  
Returns the parameter expression of the argument.

Returns:  Parameter expression

Return type:  str

GetType()[¶](#ApiClient.EdiabasArgument.GetType "Link to this definition")  
Returns the type of the argument.

Returns:  Type of the argument

Return type:  str

SetParameterExpression(*[expr](#ApiClient.EdiabasArgument.SetParameterExpression.expr "ApiClient.EdiabasArgument.SetParameterExpression.expr (Python parameter) — Parameter expression")*)[¶](#ApiClient.EdiabasArgument.SetParameterExpression "Link to this definition")  
Sets the parameter expression of the argument. Mind that by changing parameters of arguments the result list may change as well. Make sure you are working with the latest result list by calling GetResults after all arguments are set.
