*class* TsPackage[¶](#ApiClient.TsPackage "Link to this definition")  
Represents a deprecated way to call subpackages. For most test cases those steps are migrated automatically. If not, this facade will be returned when reading the contents.

Deprecated:  
Deprecated since version 2023.3: Please use `CreateTsPackageCall` instead. See [`MappingApi.CreatePackageMappingItem`](../MappingApi.md#ApiClient.MappingApi.CreatePackageMappingItem "ApiClient.MappingApi.CreatePackageMappingItem (Python method) — Creates a package mapping item referencing the given package path. For example, CreatePackageMappingItem('MyPackage.pkg') results in a reference to '<workspace.packages>/MyPackage.pkg'.") for creating a package mapping item.
