# TsReport[¶](#tsreport "Link to this heading")

*class* TsReport[¶](#ApiClient.TsReport "Link to this definition")  
Represents a report test step.

When executed within a test case, a report test step creates a report entry with the configured information. These are:

- Column texts for action, name, value, and comment

- A rating

- A file attachment

- A comment

- A table

The type of the table cells is string. Each cell contains a Python expression. The expressions are evaluated when the test step is executed. The cells of the table in the report will contain the resulting values.

Returned by

- [`TestStepApi.CreateTsReport`](../TestStepApi.md#ApiClient.TestStepApi.CreateTsReport "ApiClient.TestStepApi.CreateTsReport (Python method) — Creates a new report test step")
