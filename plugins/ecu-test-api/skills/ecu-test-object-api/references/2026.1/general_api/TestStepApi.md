callMode : str[¶](#ApiClient.TestStepApi.CreateTsCall.callMode "Permalink to this definition")  
Have the test step use this call method. Possible values are:

- ’CALL’ - actively call the target function (default value, not supported by [`DiagDidMappingItem`](MappingApi/DiagDidMappingItem.md#ApiClient.DiagDidMappingItem "ApiClient.DiagDidMappingItem (Python class) — MappingApi.CreateDiagDidMappingItem"))

- ’READ_PARAMS’ - listen to someone else calling the target function and read the input parameters of that call (only supported for [`ServiceMappingItem`](MappingApi/ServiceMappingItem.md#ApiClient.ServiceMappingItem "ApiClient.ServiceMappingItem (Python class) — MappingApi.CreateServiceMappingItem"))

- ’READ_RETURNS’ - listen to someone else calling the target function and read the return values of that call (only supported for [`ServiceMappingItem`](MappingApi/ServiceMappingItem.md#ApiClient.ServiceMappingItem "ApiClient.ServiceMappingItem (Python class) — MappingApi.CreateServiceMappingItem"))

- ’READ_EVENT’ - read the parameters of the target event for the last time it was sent (only supported for [`ServiceMappingItem`](MappingApi/ServiceMappingItem.md#ApiClient.ServiceMappingItem "ApiClient.ServiceMappingItem (Python class) — MappingApi.CreateServiceMappingItem"))

- ’SEND_EVENT’ - send the target event with the specified paramters (only supported for [`ServiceMappingItem`](MappingApi/ServiceMappingItem.md#ApiClient.ServiceMappingItem "ApiClient.ServiceMappingItem (Python class) — MappingApi.CreateServiceMappingItem"))

- ’KWP_LONG_TERM_ADJUSTMENT’ - adjust the value of a DID (only for KWP2000) (only supported for [`DiagDidMappingItem`](MappingApi/DiagDidMappingItem.md#ApiClient.DiagDidMappingItem "ApiClient.DiagDidMappingItem (Python class) — MappingApi.CreateDiagDidMappingItem"))
